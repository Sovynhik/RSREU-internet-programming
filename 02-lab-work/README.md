## Задание
Внедрить JavaScript в ранее созданный сайт тремя способами: схема URL, обработчики событий, контейнер `<script>`. По два примера на каждый способ.

## Реализация
Модифицирована главная страница `index.html`. Добавлены:

1. **Схема URL** – ссылки с `javascript:`:
    - «Поздороваться» – вызов `showCustomAlert('Привет!')`
    - «Случайный факт» – вызов `showRandomFact()`

2. **Обработчики событий** – кнопки с `onclick`:
    - ☀️ – дневная тема (сброс цветов)
    - 🌙 – ночная тема (тёмный фон)

3. **Контейнер `<script>`**:
    - Определены функции (`showCustomAlert`, `closeModal`, `showRandomFact`, `setSunTheme`, `setMoonTheme`)
    - Изменён цвет заголовка и текст подзаголовка при загрузке

Добавлено модальное окно (стилизованный блок), которое появляется при вызове функций.
