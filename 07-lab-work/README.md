## Задание
Разработать программу для анализа трёх функций:
- **f₁(x)** = x·√(sin³(x+10)) + (x³ – cos x)/x
- **f₂(x)** = sin² x – |sin(x–4)|
- **f₃(x)** = e^(x–2) + x³ + 2x·ln(x+3)/7

Для каждой функции на отрезке [a, b] с шагом h вычислить характеристики:
1. Минимум функции.
2. Проверка, является ли функция монотонно убывающей.
3. Количество нулевых значений (с погрешностью 1e-10).

Необходимо реализовать возможность применения к функциям обёрток (функций высшего порядка), реализующих:
- мемоизацию (кэширование результатов),
- отладочный вывод в консоль (время, аргумент, значение),
- подсчёт количества вызовов.

Характеристики вычисляются отдельными функциями, передаваемыми в общую функцию расчёта в виде массива. Результат возвращается также массивом.

## Реализация
Программа состоит из трёх файлов:
- `index.html` – структура страницы с элементами управления (ввод a, b, h, выбор функции, чекбоксы для характеристик и обёрток).
- `style.css` – оформление в единой цветовой гамме (синие тона, скругления, тени).
- `script.js` – логика вычислений, обёртки, отрисовка результатов.

### Основные функции JavaScript
- **f1, f2, f3** – исходные математические функции (обрабатывают особые случаи, возвращают NaN).
- **calcMin(y)** – поиск минимума среди конечных значений.
- **isMonotoneDecreasing(x, y)** – проверка монотонного убывания (с пропуском NaN).
- **countZeros(y)** – подсчёт значений, близких к нулю.
- **wrapFunction(f, options)** – функция высшего порядка, возвращающая обёрнутую функцию с методами доступа к кэшу и счётчику. Реализует мемоизацию, отладку и подсчёт вызовов.
- **computeCharacteristics(func, a, b, h, charFuncs)** – вычисляет характеристики для заданной функции на сетке.
- Обработчик кнопки «Вычислить» – считывает параметры, создаёт обёртку, вычисляет характеристики и отображает результаты, размер кэша, количество вызовов и таблицу кэша.

### Интерфейс
- Поля ввода: a, b, h.
- Выпадающий список для выбора функции.
- Чекбоксы для выбора характеристик (минимум, монотонность, нули).
- Чекбоксы для выбора обёрток (мемоизация, отладка, счётчик).
- Кнопка «Вычислить».
- Блоки вывода: результаты характеристик, информация о мемоизации/счётчике, таблица кэша.
