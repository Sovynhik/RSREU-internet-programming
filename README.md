# Проект «Смешарики» – Лабораторные работы по JavaScript

Данный проект представляет собой серию лабораторных работ по курсу программирования на JavaScript. Все работы выполнены на единую тему – популярный российский мультфильм **«Смешарики»**. Каждая последующая лабораторная расширяет предыдущую, добавляя новые возможности и изучая различные аспекты языка.

---

## Состав проекта

### Лабораторная работа №1 – Статический сайт (HTML/CSS)
- **Файлы:** `index.html`, `characters.html`, `world.html`, `facts.html`, `styles.css`
- **Описание:** Создан четырёхстраничный сайт с навигацией, списками, таблицей и изображениями. Оформлен в едином стиле (синяя гамма, скругления).
- **Основные элементы:** главная страница с историей и наградами, страница персонажей с описанием и галереей, страница мира с домами героев, страница фактов с таблицей.

### Лабораторная работа №2 – Внедрение JavaScript
- **Файлы:** модифицирован `index.html`, добавлен `script.js` (встроенный)
- **Описание:** В главную страницу добавлены три способа использования JS:
    - схема URL (ссылки «Поздороваться» и «Случайный факт»);
    - обработчики событий (кнопки смены темы ☀️/🌙);
    - встроенный скрипт с функциями и модальным окном.
- **Новые возможности:** модальное окно с приветствием, случайные факты, смена цветовой темы.

### Лабораторная работа №3 – Табулирование функции двух переменных (вариант 16)
- **Файлы:** `function-tabulation.html`, `tabulation.js`, `style.css` (обновлён)
- **Описание:** Программа для вычисления функции f(x, y) на заданных диапазонах с шагом, поиска минимума и максимума. Результат выводится в виде HTML-таблицы с подсветкой экстремумов.
- **Особенности:** реализована функция вычисления факториала, обработка NaN и бесконечностей, адаптивный интерфейс.

### Лабораторная работа №4 – Динамическая таблица с действиями (вариант 14)
- **Файлы:** `index.html` (новая версия), `script.js`, `style.css`
- **Описание:** Создание динамической таблицы случайных чисел. Реализованы три действия:
    1. заполнение красных ячеек (центральные столбцы) суммой или средним;
    2. удаление цвета для указанных строк;
    3. поворот таблицы на 90° вправо.
- **Техника:** только общие методы DOM (createElement, appendChild, removeChild), без табличных методов.

### Лабораторная работа №5 – Матрицы и массивы (вариант 4)
- **Файлы:** `index.html`, `script.js`, `style.css` (новая версия)
- **Описание:** Генерация двух матриц M1 и M2. Реализована функция, перемещающая в каждой строке чётные элементы в начало, нечётные – в конец. Отображаются исходные и преобразованные матрицы.
- **Использование:** встроенные методы массивов (map, forEach, sort), мутабельный подход.

### Лабораторная работа №6 – Массив объектов «Детали» (вариант 4)
- **Файлы:** `index.html`, `script.js`, `style.css`
- **Описание:** Массив объектов с информацией о деталях (название, габариты, материал, масса, поставщики). Реализованы:
    - удаление свойства «материал» при массе ниже порога;
    - добавление свойства «габаритный объём»;
    - сортировка по убыванию массы;
    - сохранение/загрузка JSON.
- **Особенности:** сериализация, работа с вложенными объектами и массивами.

### Лабораторная работа №7 – Анализ функций с мемоизацией (вариант 4)
- **Файлы:** `index.html`, `script.js`, `style.css`
- **Описание:** Анализ трёх математических функций на отрезке [a,b] с шагом h. Характеристики: минимум, монотонность, количество нулей. Функции высшего порядка:
    - мемоизация (кэширование результатов);
    - отладочный вывод в консоль;
    - подсчёт количества вызовов.
- **Интерфейс:** выбор функции, характеристик, обёрток, вывод таблицы кэша.

### Лабораторная работа №8 – Класс «Квадратная матрица» (вариант 4)
- **Файлы:** `index.html`, `script.js`, `style.css`
- **Описание:** Реализация классов через прототипы. Базовый класс `BaseObject` с логированием вызовов. Класс `Matrix` с методами доступа к элементам, сложением, вычитанием, умножением, поиском максимума/минимума. Переопределён `toString()`.
- **Взаимодействие:** создание двух матриц A и B, выполнение операций, просмотр лога в консоли.

## Технологии

- Принципы: прототипное наследование, функции высшего порядка, замыкания, мемоизация, сериализация JSON, работа с DOM.

## Инструкция по запуску
1. Скачайте все файлы проекта.
2. Убедитесь, что сохранена структура папок (изображения лежат в папке `../images/` относительно HTML-файлов). При необходимости скорректируйте пути к изображениям в HTML.
3. Откройте любой `.html` файл в современном браузере (Chrome, Firefox, Edge).
4. Для корректной работы JSON-загрузки в лабораторной №6 используйте локальный сервер (например, `Live Server` в VS Code), так как браузер может блокировать загрузку файлов из-за политики CORS.
