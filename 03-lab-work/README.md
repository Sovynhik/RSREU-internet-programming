# Лабораторная работа №3 – Табулирование функции двух переменных

**Тема:** Табулирование функции f(x, y) с нахождением минимума и максимума  
**Вариант 16**  
**Выполнил:** Савушкин Д.А., группа 244

---

## Задание
Реализовать программу для табулирования функции двух переменных f(x, y) на заданных диапазонах с заданным шагом. Функция задана выражением:

f(x, y) = (x/2) · ∏(1 + y·xⁿ/n!) + (x+y)/(x²+1) · ∏(y + xⁿ⁺¹/(2n-5))

где первое произведение берётся по n от 1 до nm1, второе – по n от 1 до nm2. Параметры nm1 и nm2 – целые числа в диапазоне [2, 6]. Необходимо вычислить значения функции во всех точках сетки, найти максимальное и минимальное значение и выделить их в таблице. Вывод результатов осуществляется в виде HTML-таблицы.

## Реализация
Программа состоит из трёх файлов:
- `function-tabulation.html` – структура страницы с формой ввода параметров.
- `style.css` – оформление в едином стиле с предыдущими работами.
- `tabulation.js` – логика вычислений и отрисовки.

### Основные функции JavaScript
- **factorial(n)** – вычисляет факториал целого неотрицательного числа. Используется при расчёте произведений.
- **tabulate(nm1, nm2, xStart, xEnd, xStep, yStart, yEnd, yStep)** – выполняет табулирование:
   - Формирует массивы значений x и y с учётом шага.
   - Для каждой пары (x, y) вычисляет значение функции по формуле.
   - Обрабатывает возможные исключения (деление на ноль, NaN, бесконечность).
   - Находит максимальное и минимальное значение среди допустимых результатов.
   - Возвращает объект с массивами координат, матрицей значений и информацией об экстремумах.
- **renderTable(data, container)** – строит HTML-таблицу по полученным данным:
   - Заголовок с фиксированными значениями y.
   - Каждая строка соответствует значению x.
   - Ячейки с максимумом и минимумом выделяются цветом (зелёный и красный).
   - При наличии неопределённых значений выводится «---».

### Интерфейс
На странице расположены три группы полей ввода:
- Параметры функции: nm1 и nm2 (целые числа от 2 до 6).
- Диапазон по x: начальное значение, конечное, шаг.
- Диапазон по y: аналогично.

Кнопка «Табулировать» запускает вычисления. Результат отображается в виде таблицы с информацией о максимуме и минимуме сверху.

При загрузке страницы автоматически выполняется табулирование с параметрами по умолчанию.

### Использованные элементы
- Формы ввода (input type="number") с ограничениями.
- Обработчики событий (addEventListener) для кнопки и загрузки окна.
- Динамическое создание таблицы через innerHTML.
- Стилизация с использованием CSS (flexbox, тени, скругления).
- Выделение экстремумов классами `.max-cell` и `.min-cell`.
