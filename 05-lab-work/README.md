## Задание
Составить функцию, которая перемещает в начало вектора все чётные элементы, а в конец – нечётные. Используя эту функцию, преобразовать все строки матриц **M1** и **M2**. Размеры матриц и диапазон случайных чисел задаются пользователем (строки и столбцы от 1 до 10). Все операции с массивами выполняются встроенными методами `Array` (без циклов `for`/`while`).

## Реализация
Программа состоит из трёх файлов:
- `index.html` – структура страницы с формой ввода параметров (размеры M1, M2, диапазон) и четырьмя блоками для отображения матриц.
- `style.css` – оформление в едином стиле (синяя гамма, скругления, тени).
- `script.js` – логика генерации, преобразования и отрисовки.

### Основные функции JavaScript
- **generateMatrix(rows, cols, min, max)** – создаёт матрицу случайных чисел с помощью `Array.from`.
- **renderMatrix(matrix, container)** – строит HTML-таблицу через `createElement`, `appendChild` (очистка через `removeChild`).
- **reorderEvenOdd(arr)** – преобразует массив на месте, перемещая чётные элементы в начало, нечётные в конец. Использует `sort` с компаратором, сохраняющим относительный порядок внутри групп.
- **transformMatrix(matrix)** – применяет `reorderEvenOdd` к каждой строке матрицы через `forEach`.
- Обработчики кнопок: «Сгенерировать» создаёт матрицы, «Преобразовать» создаёт копии (`map` + `slice`) и применяет преобразование, затем отображает результат.

Все операции с массивами выполнены исключительно встроенными методами, мутабельный подход соблюдён.

## Использованные элементы
- Генерация случайных чисел через `Math.random`.
- Создание матриц без циклов: `Array.from({ length: rows }, () => Array.from(...))`.
- Копирование строк: `row.slice()`.
- Сортировка с пользовательским компаратором.
- Динамическое создание таблиц через DOM.
- Обработка событий кнопок.
