# Лабораторная работа №1 – Создание статического сайта

**Тема:** Мультфильм «Смешарики»  
**Выполнил:** Савушкин Д.А., группа 244

---

## Задание
Разработать 4-страничный сайт (главная + три подстраницы) на произвольную тему с использованием HTML и CSS.

## Реализация
Сайт состоит из страниц: `index.html` (главная), `characters.html` (персонажи), `world.html` (мир), `facts.html` (факты).  
Все страницы объединены единым стилем (`styles.css`) и навигационным меню.

### Использованные элементы
- **Списки** – маркированные и нумерованные (награды, особенности мира, факты).
- **Таблица** – на странице фактов (основные данные о мультфильме).
- **Изображения** – логотип, галерея персонажей (9 картинок), иллюстрация мира.
- **Форматирование текста** – заголовки разных уровней, выделение имён жирным.
- **Оформление** – цветовая схема (синие тона), эффекты при наведении на ссылки и картинки.

## Структура
- **Главная** – общая информация, история, награды.
- **Персонажи** – описания героев + галерея.
- **Мир** – описание Ромашковой долины и домов.
- **Факты** – таблица и интересные сведения.
