## Задание
Сформировать динамическую таблицу с произвольным размером (от 1 до 50) с помощью методов манипулирования DOM. Реализовать три действия над оригинальной таблицей:
1. Заполнить красные ячейки (центральные столбцы) суммой или средним арифметическим всех чисел таблицы.
2. Убрать цветовую заливку для строк, номера которых введёт пользователь через запятую.
3. Повернуть таблицу на 90 градусов вправо.

### Основные функции JavaScript
- **getValidatedDimensions()** – проверяет и корректирует введённые размеры.
- **generateRandomData(rows, cols)** – создаёт матрицу случайных чисел от 1 до 100.
- **getCentralIndices(length)** – возвращает индексы центральных элементов (один или два) для массива.
- **renderTable(data, options)** – строит таблицу с использованием `createElement`, `appendChild`, `removeChild`. Применяет классы `red-cell` и `blue-cell` для центральных столбцов и строк. Учитывает опции `overrideRed` (заполнение красных ячеек) и `excludeRows` (исключение строк из раскраски).
- **createNewTable()** – создаёт новую таблицу и сохраняет оригинальные данные.
- Обработчики событий для кнопок действий.

### Действия
1. **Заполнение красных ячеек** – вычисляется общая сумма или среднее, затем вызывается `renderTable` с опцией `overrideRed`. Красными считаются центральные столбцы.
2. **Убрать цвет для строк** – парсятся введённые номера строк (1-индексация), формируется массив исключаемых индексов. При повторной отрисовке этим строкам не назначаются цветные классы.
3. **Поворот таблицы** – создаётся новая матрица размером `cols × rows`, где элемент `[i][j]` повёрнутой таблицы равен исходному `[rows-1-j][i]`. Затем вызывается `renderTable` для новой матрицы, цвета пересчитываются автоматически.

### Использование DOM
- Создание элементов: `createElement('table')`, `createElement('tr')`, `createElement('td')`.
- Добавление: `appendChild`, удаление: `removeChild` (очистка контейнера).
- Манипуляция классами через `classList.add`.
- Специализированные табличные методы (`insertRow`, `deleteRow`, `rows`, `cells`) **не используются** – условие выполнено.
