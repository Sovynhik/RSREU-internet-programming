## Задание
Разработать программу для работы с массивом объектов «Детали». Каждый объект содержит:
- наименование детали;
- габаритные размеры (отдельный объект с полями длина, ширина, высота);
- материал;
- масса детали;
- список поставщиков (массив объектов с названием организации и телефоном).

Необходимо реализовать три действия:
1. Удаление свойства «материал» у деталей с массой меньше заданного порога.
2. Добавление свойства «габаритный объём» (произведение длины, ширины, высоты).
3. Сортировка массива по убыванию массы (с созданием отдельного отсортированного массива).

Программа должна позволять задавать количество деталей, порог массы, диапазон случайных чисел, генерировать данные, выполнять действия и выводить результаты в виде таблиц. Также необходима сериализация массива в JSON и загрузка из JSON.

## Реализация
Программа состоит из трёх файлов:
- `index.html` – структура страницы с формой ввода параметров и четырьмя блоками для отображения таблиц.
- `style.css` – оформление в едином стиле (синяя гамма, скругления, тени, адаптивная сетка).
- `script.js` – логика генерации данных, обработки действий, отрисовки таблиц и работы с JSON.

### Основные функции JavaScript
- Генерация случайных чисел и телефонов.
- **createDetail(index)** – создание объекта детали со случайными параметрами.
- **generateDetails(count)** – создание массива деталей.
- **formatSuppliers(suppliers)** – отображение первого поставщика и количества остальных.
- **renderDetails(detailsArray, container)** – построение HTML-таблицы через DOM (createElement, appendChild, removeChild).
- **renderAll()** – обновление всех таблиц.

### Действия
- **«Удалить материал»** – удаление свойства `material` у деталей с массой ниже порога (мутабельно, с подсчётом).
- **«Добавить габаритный объём»** – добавление свойства `volume` (длина × ширина × высота) для всех деталей.
- **«Сортировать по убыванию массы»** – создание глубокой копии массива и сортировка по `mass` (убывание).
- **«Сохранить в JSON»** – экспорт исходного массива в файл `details.json`.
- **«Загрузить из JSON»** – импорт данных из файла и обновление таблиц.

Все операции выполняются с использованием встроенных методов массивов и объектов (`forEach`, `map`, `sort`, `slice`, `delete`), мутабельный подход соблюдён.

## Использованные элементы
- Генерация случайных данных.
- Работа с объектами и массивами объектов.
- Создание HTML-таблиц через DOM.
- Сериализация и десериализация JSON.
- Стилизация интерфейса с помощью CSS (flexbox, grid, тени, скругления).
